# Improvements

## Реализовано

1. Явный отказ без выдумок.

   Генератор проверяет `MIN_SCORE`. Если найденные чанки ниже порога, он возвращает отказ вместо ответа по слабому контексту.

2. Воспроизводимая подготовка данных.

   `scripts/prepare_datasets.py` может заново собрать `datasets.json` из архива RusLit на GitHub. При этом готовый датасет уже лежит в репозитории, поэтому обычный запуск не зависит от сети.

3. Проверяемый retrieval.

   Есть отдельные консольные проверки `check_retrieval.py` и `check_generator.py`, а также тест retriever на временном индексе.

## План дальнейших улучшений

- Добавить reranking найденных чанков.
- Сделать оценочный набор вопросов с ожидаемыми источниками.
- Добавить подсветку совпавших слов в Streamlit UI.
