# Соглашения

## Данные

- `data/raw/datasets.json` - список объектов.
- Обязательные поля: `doc_id`, `title`, `source`, `text`.
- `doc_id` уникален и стабилен.
- В индекс попадает только `text`.

## Артефакты pipeline

- `data/processed/documents.jsonl` - результат ingest.
- `data/processed/chunks.jsonl` - чанки.
- `data/index/vectorizer.pkl` и `data/index/matrix.npz` - TF-IDF индекс.
- `data/processed/` и `data/index/` генерируются и не коммитятся.

## Код

- Пути хранятся в `app/config.py`.
- Скрипты добавляют корень проекта в `sys.path`, чтобы запускаться из CLI.
- Тесты не требуют заранее построенного индекса, кроме проверки контракта исходного датасета.

## Ответ

- Ответ строится только из найденных фрагментов.
- Источники показываются независимо от того, был ли сформирован ответ.
- При низком score возвращается отказ.
