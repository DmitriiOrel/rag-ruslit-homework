# Vision и границы MVP

## Пользователь

Студент, которому нужно понять, как устроен RAG pipeline: ingest, chunking, index, retrieval, answer и UI.

## MVP

- Локальный запуск через `uv`.
- Данные уже лежат в репозитории в `data/raw/datasets.json`.
- Индекс пересобирается одной командой.
- Ответ содержит цитируемые источники.
- Отказ срабатывает, если релевантного контекста нет.

## Не входит в MVP

- Внешняя LLM.
- Векторные embeddings.
- Авторизация, база данных, production deployment.
- Полнотекстовый анализ всех метаданных Gutenberg.

## Почему такой стек

TF-IDF и Streamlit достаточны для учебного проекта: они быстро запускаются, легко проверяются тестами и прозрачно показывают, почему найден тот или иной фрагмент.
